# Telegram Steganography Bot (aiogram)

Коротко: цей бот на `aiogram` дає змогу обрати один із чотирьох прототипів та виконати операції `Encode` або `Decode`.

Особливості:
- Потрібно надсилати зображення як `Document` (щоб уникнути стиснення Telegram). Якщо надіслано як `photo`, бот попросить повторити як `Document`.
- Після `Encode` бот повертає файл-результат як `Document` (без стиснення).
- Для деяких прототипів потрібен числовий ключ — бот запитає його при потребі.

Запуск:
1. Встановіть залежності: `pip install -r requirements.txt`
2. Встановіть змінну оточення `TG_BOT_TOKEN` або відредагуйте `telegram_bot.py`, вставивши токен у змінну `API_TOKEN`.
3. Запустіть бот: `python telegram_bot.py`

Розташування файлів:
- Всі тимчасові файли зберігаються у `tmp/<user_id>/`.
- Бот імпортує ваші модулі `Prototype_1..4` прямо з робочої директорії, тому не змінюйте структуру папок.

Примітки по безпеці:
- Це простий PoC. Не зберігайте довготривало тимчасові файли у відкритому вигляді на сервері.
